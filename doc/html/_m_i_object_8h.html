Detailed Description

Basic object which serves as a basic block for all other objects in the system.

The Object is the pseudo-object class that provides fundamental methods that are needed by all Bitcoin objects It handles reference handling (incrementing and decrementing the number of references to a Object) and the creation and cleaning up of memory
