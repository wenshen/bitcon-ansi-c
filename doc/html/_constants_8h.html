enum NetworkCommunicatorFlags
Enumerator:
NETWORK_COMMUNICATOR_AUTO_HANDSHAKE 

Automatically share version and verack messages with new connections.

NETWORK_COMMUNICATOR_AUTO_DISCOVERY 

Automatically discover peers and connect up to the maximum allowed connections using the supplied Version. This involves the exchange of version messages and addresses.

NETWORK_COMMUNICATOR_AUTO_PING 

Send ping messages every "heartBeat" automatically. If the protocol version in the Version message is 60000 or over, bitcoin will use the new ping-pong specification.
