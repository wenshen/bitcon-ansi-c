Definition at line 23 of file NetworkAddress.h.

ByteArray* ip

IP address. Should be 16 bytes for the IPv6 compatible format. The NetworkAddress should have exclusive access to the ByteArray to avoid potential threading issues as the NetworkAddresses are protected by mutexes maintained by NetworkCommunicators but the ip is not.

Definition at line 28 of file NetworkAddress.h.

bool public

If true the address is public and should be relayed. If true, upon a lost or failed connection, return to addresses list. This doesn't include failure from "CBNetworkCommunicatorConnect". If false the address is private and should be forgotten when connections are closed.
