BigInt decodeBase58 ( char *  str)

Decodes base 58 string into byte data as a BigInt.

Parameters:
strBase 58 string to decode.
Returns:
Pointer to the byte data as a BigInt. The byte data will be created in this function. Remember to free the data. On error the big int will have a NULL data pointer.
