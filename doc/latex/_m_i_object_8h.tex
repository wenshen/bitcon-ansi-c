\subsection{Detailed Description}
Basic object which serves as a basic block for all other objects in the system. The \hyperlink{struct_m_i_object}{Object} is the pseudo-\/object class that provides fundamental methods that are needed by all Bitcoin objects It handles reference handling (incrementing and decrementing the number of references to a \hyperlink{struct_m_i_object}{Object}) and the creation and cleaning up of memory
